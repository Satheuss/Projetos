Esse projeto cria uma visualização 3D de uma função matemática que gera ondas circulares a partir do centro,
parecidas com o efeito de uma pedra caindo na água.
Para isso, o código monta uma grade de pontos nos eixos X e Y,
calcula o valor da função seno da distância desses pontos até o centro e usa esses valores para formar uma superfície ondulada.
Por fim, essa superfície é exibida como um gráfico 3D colorido na tela.
