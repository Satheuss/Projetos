App de Imagens Aleatórias

Aplicação em Python que exibe imagens aleatórias da internet usando Tkinter, pois é nativo e não precisa ser instalado.
O app tem uma biblioteca de pouco mais de 20 imagens, que provavelmente será expandida no futuro próximo.
App feito para estudo de introdução ao Tkinter.

Requisitos para rodar:
pip install pillow requests
