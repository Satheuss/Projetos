O projeto se resumo em um jogo interativo no terminal que simula a roleta russa usando números aleatórios para decidir se o jogador sobrevive ou perde.
Ele usa a biblioteca random para simular posições aleatórias do tambor e da bala.
O jogo funciona em até 6 rodadas,
se sobreviver as 6 você ganha.
Não precisa de nenhum pré requisito para rodar o código.
